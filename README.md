sd_midiplay
===
Android6.xのmedia.midiを使ったサンプルプログラム  
16ステップのシーケンサで、CH10からハイハットの音を出力。  

必要なもの  
- USB OTGケーブル
- USBケーブル(かmicroUSBケーブル)
- USB接続のMIDIインターフェースかシンセ

使い方
- 音を出したい位置を指定して、PLAYボタンを押す
- STOPを押すと一時停止、もう一度押すと頭に戻る
- テンポは5段階で切換

ビルド  
Android Studioで  
New→Project from Version Control  
New→Import Project  
のどちらかで、プロジェクトを読み込み
